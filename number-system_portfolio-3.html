Number System

What are Number Systems?

- In computer science, are methods for representing numeric values within a computer.

- Number systems are fundamental in computing, as they define how data is represented and manipulated.

- They are essential for the functioning of computers as they enable data storage, processing, and communication.

Types of Number Systems

  • Binary Number System (Base-2)
  • Decimal Number System (Base-10)
  • Octal Number System (Base-8)
  • Hexadecimal Number System (Base-16)

Binary (Base-2)

- The binary system uses only two digits, 0 and 1, and is the backbone of digital computing.

- Computers process data in binary, where every bit represents a power of 2.

- This system is efficient for representing machine-level instructions and data processing in CPUs and memory systems.

Computers operate using this system because transistors in digital circuits can exist in two states, corresponding to 0 and 1.

All data, whether numbers, text, or instructions, are processed in binary format at the hardware level.

(Princeton University, 2019)

Decimal (Base-10)

- This is the most familiar number system to humans, using digits from 0 to 9.

- Although not directly used by computers, the decimal system is often converted to binary or hexadecimal for processing.

- In computing, it is often used in user interfaces and software outputs. However, internally, decimal numbers are converted to binary for processing.

(Washington State University, 2020)

Octal (Base-8)

- This system uses digits 0 through 7 and is often used in computer science as a more compact way of representing binary numbers.

- Each octal digit represents three binary digits

- It was historically popular in early computing systems due to its easy conversion from binary

(Harle, University of Cambridge, 2020)

Hexadecimal (Base-16)

- The hexadecimal system, using digits 0-9 and letters A-F, used in computer programming and memory addressing is a more human-readable way of representing binary data.

- It is a more human-readable than binary. Useful for compactly representing large binary numbers

- Each hexadecimal digit corresponds to four binary digits.

Hexadecimal is widely used in programming, memory addressing, and debugging, providing a compact and readable form of binary data

(Princeton University, 2019)
